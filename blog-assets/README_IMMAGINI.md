### Formato e Dimensioni
- **Formato:** WebP per web (ottimizzato)
- **Dimensione copertina:** 1200x630px (ratio 1.91:1 per social)
- **Dimensione immagini interne:** 800x600px o 1200x800px
- **Compressione:** Qualità 80-85%

### Ottimizzazione
Convertire in WebP con:
```bash
# Mac
cwebp -q 80 input.jpg -output.webp

# Online
https://convertio.co/it/webp-converter/
```
